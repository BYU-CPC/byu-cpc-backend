# Local development quick start

## Get access

Get access to the firebase console: https://console.firebase.google.com/project/byu-cpc/settings/iam

## Installation

Make sure you have installed:
- PostgreSQL
- Python 3
- Google Cloud SDK (`gcloud`)

Run the setup script:

```bash
./scripts/setup.sh
```

Then, run:

```bash
python3 main.py
```
